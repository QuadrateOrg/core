# CosmWasm support

This package contains CosmWasm integration points.

This package provides first class support for:

- Queries
  - Denoms
  - Pools
  - Prices
- Messages / Execution
  - Minting / controlling of new native tokens
  - Swap


## Command line interface (CLI)

- Commands

```sh
  quadrated tx wasm -h
```

- Query

```sh
  quadrated q wasm -h
```
